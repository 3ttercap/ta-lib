# ta-lib.stddev

The ta-lib method `stddev` exported as a module.

## Installation

Using npm:
```bash
$ npm i --save ta-lib.stddev
```

In Node.js:
```js
var stddev = require('ta-lib.stddev')
```

See [package source](https://github.com/WaiSiuKei/ta-lib/tree/master/ta-lib.stddev) for more details.
